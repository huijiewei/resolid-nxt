# Divider

Divide the content areas and separate the modules.

## Import

```javascript
import { Divider } from '@resolid/nxt-ui';
```

## Usage

<Demo file={'DividerDemo.tsx'} demo={'Usage'} height={288} />

## Variants

Divider has 3 variant styles you can use. Pass the `variant` prop and use either `solid`, `dotted` or `dashed`.

<Demo file={'DividerDemo.tsx'} demo={'Variants'} height={218} />

## Colors

Change the color of the dividers by passing the `color` prop. Divider supports `primary`, `success`, `warning`, `danger`, and `neutral` colors.

<Demo file={'DividerDemo.tsx'} demo={'Colors'} height={300} />

## Sizes

Divider use `${size}px` set size, size should is number.
