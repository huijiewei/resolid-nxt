# Spinner

Used for the loading state of pages. When part of the page is waiting for asynchronous data or is in the rendering process, appropriate loading amimation will effectively alleviate user anxiety.

## Import

```javascript
import { Spinner } from '@resolid/nxt-ui';
```

## Colors

Change the color of the spinner by passing the `color` prop. Spinner supports `primary`, `success`, `warning`, `danger`, and `neutral` colors.

<Demo file={'SpinnerDemo.tsx'} height={211} demo={'Colors'} />

## Sizes

Spinner has 5 sizes you can use. Pass the `size` prop and use either `xs`, `sm`, `md`, `lg` or `xl`.
