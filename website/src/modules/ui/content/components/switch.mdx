# Switch

A mutually exclusive control that allows the user to enable or disable a certain feature.

## Import

```javascript
import { Switch } from '@resolid/nxt-ui';
```

## Colors

Change the color of the switch by passing the `color` prop. Switch supports `primary`, `success`, `warning`, `danger`, and `neutral` colors.

<Demo file={'SwitchDemo.tsx'} height={371} demo={'Colors'} />

## Sizes

Switch has 5 sizes you can use. Pass the `size` prop and use either `xs`, `sm`, `md`, `lg` or `xl`.
