# Avatar

The Avatar component is used to represent user, and displays the profile picture, initials or fallback.

## Import

Resolid NxtUI exports 3 avatar-related components:

- Avatar: The image that represents the user.
- AvatarBadge: A wrapper that displays its content on the right corner of the avatar.
- AvatarGroup: A wrapper to stack multiple Avatars together.


```javascript
import { Avatar, AvatarGroup, AvatarBadge } from '@resolid/nxt-ui';
```
