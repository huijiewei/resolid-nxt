# Checkbox

In a set of data, users can select one or more data points by checking checkboxes.

## Import

```javascript
import { Checkbox, CheckboxGroup } from '@resolid/nxt-ui';
```

## Usage

<Demo file={'CheckboxDemo.tsx'} height={310} demo={'Usage'} />

## Colors

Change the color of the checkbox by passing the `color` prop. Checkbox supports `primary`, `success`, `warning`, `danger`, and `neutral` colors.

<Demo file={'CheckboxDemo.tsx'} height={367} demo={'Colors'} />

## Sizes

Checkbox has 5 sizes you can use. Pass the `size` prop and use either `xs`, `sm`, `md`, `lg` or `xl`.
