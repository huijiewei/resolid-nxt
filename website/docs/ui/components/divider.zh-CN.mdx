---
title: 分割线
description: 分隔内容区域并分离模块。
---

## 导入

```javascript
import { Divider } from '@resolid/nxt-ui';
```

## 用法

<Demo file={'DividerDemo.tsx'} demo={'Usage'} height={288} />

## 变体

分割线有三种不同的样式变体可供使用。通过传递 `variant` 属性，可以选择使用 `solid`、`dotted` 或 `dashed` 中的任意一种。

<Demo file={'DividerDemo.tsx'} demo={'Variants'} height={218} />

## 颜色

通过传递 `color` 属性来更改分隔线的颜色。分隔线支持 `primary`、`success`、`warning`、`danger` 和 `neutral` 这些颜色。

<Demo file={'DividerDemo.tsx'} demo={'Colors'} height={300} />

## 大小

分隔线使用 `${size}px` 来设置大小，大小应为数字。

<Demo file={'DividerDemo.tsx'} demo={'Sizes'} height={221} />
