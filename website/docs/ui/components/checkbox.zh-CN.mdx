title: 复选框
description: 在一组数据中，用户可以通过选中复选框来选择一个或多个数据点。
---

## 导入

```javascript
import { Checkbox, CheckboxGroup } from '@resolid/nxt-ui';
```

## 颜色

通过传递 `color` 属性来更改复选框的颜色。复选框支持 `primary`、`success`、`warning`、`danger` 和 `neutral` 这些颜色。

<Demo file={'CheckboxDemo.tsx'} height={367} demo={'Colors'} />

## 大小

复选框有五种可用的尺寸。通过传递 `size` 属性，可以选择使用 `xs`、`sm`、`md`、`lg` 或 `xl` 中的任意一种。
