## 导入

```javascript
import { Image } from '@resolid/nxt-ui';
```

## 基础

图片的属性与HTML图像元素的属性相同，可以在 [MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/img#attributes) 中找到。

图片的样式可以使用非常灵活的 [Tailwind CSS](https://tailwindcss.com) 进行自定义。
