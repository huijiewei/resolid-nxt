---
title: 头像
description: 头像组件用于表示用户，显示个人资料图片、首字母或备用内容。
---

## 导入

```javascript
import { Avatar, AvatarGroup, AvatarBadge } from '@resolid/nxt-ui';
```

- `Avatar`: 代表用户的图片。
- `AvatarBadge`: 一个包装器，将其内容显示在头像的右上角。
- `AvatarGroup`: 一个用于将多个头像堆叠在一起的包装器。
