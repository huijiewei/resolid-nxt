---
title: 警告提示
description: 以吸引用户注意力的方式显示简短重要信息，而不会打断用户正在进行的任务。
---

## 导入

```javascript
import { Alert, AlertIcon, AlertTitle, AlertDescription } from '@resolid/nxt-ui';
```

- `Alert`: 警告提示组件的包装器。
- `AlertIcon`: 根据状态属性变化的警告提示的可视化图标。
- `AlertTitle`: 警告的标题，可以供屏幕阅读器宣读。
- `AlertDescription`: 警告的描述，可以供屏幕阅读器宣读。

## 变体

警告提示有三种不同的样式变体可供使用。通过传递 `variant` 属性，可以选择使用 `light`、`solid` 或 `outline` 中的任意一种。

<Demo file={'AlertDemo.tsx'} demo={'Variants'} height={499} />

## 颜色

通过传递 `color` 属性来更改警告提示的颜色。这将影响颜色方案和使用的图标。警告支持 `primary`、`success`、`warning`、`danger` 和 `neutral` 这些颜色。
