---
title: Breadcrumb
description: An auxiliary navigation pattern used to identify the position of a page within a hierarchical structure and allow for upward navigation as needed.
---

## Import

Resolid NxtUI exports 4 breadcrumb-related components:

- Breadcrumb: The parent container for breadcrumbs.
- BreadcrumbItem: Individual breadcrumb element containing a link and a divider.
- BreadcrumbLink: The breadcrumb link.
- BreadcrumbSeparator: The visual separator between each breadcrumb.

```javascript
import {
  Breadcrumb,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbSeparator
} from '@resolid/nxt-ui';
```
