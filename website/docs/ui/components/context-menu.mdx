---
title: ContextMenu
description: Displays a menu located at the pointer, triggered by a right-click or a long-press.
---

## Import

```javascript
import {
  ContextMenu,
  ContextMenuTrigger,
  ContextMenuContent,
  ContextMenuItem,
  ContextMenuGroup,
  ContextMenuGroupLabel,
  ContextMenuDivider,
  ContextMenuItemIndicator,
  ContextMenuCheckboxItem,
  ContextMenuRadioGroup,
  ContextMenuRadioItem,
} from '@resolid/nxt-ui';
```

- `ContextMenu`: The wrapper component provides context, state, and focus management.
- `ContextMenuTrigger`: The trigger for the menu list. Must be a direct child of `ContextMenu`.
- `ContextMenuContent`: The wrapper for the menu items. Must be a direct child of `ContextMenu`.
- `ContextMenuItem`: The trigger that handles menu selection. Must be a direct child of a `ContextMenuContent`.
- `ContextMenuGroup`: A wrapper to group related menu items.
- `ContextMenuGroupLabel`: Used to render the label of a group. It won't be focusable using arrow keys.
- `ContextMenuDivider`: A visual separator for menu items and groups.
- `ContextMenuItemIndicator`: Renders when the parent `ContextMenuCheckboxItem` or `ContextMenuRadioItem` is checked. You can style this element directly, or you can use it as a wrapper to put an icon into, or both.
- `ContextMenuCheckboxItem`: An item that can be controlled and rendered like a checkbox.
- `ContextMenuRadioGroup`: Used to group multiple `ContextMenuRadioItem`s.
- `ContextMenuRadioItem`: An item that can be controlled and rendered like a radio.
