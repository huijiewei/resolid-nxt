## Import

Resolid NxtUI exports 12 dropdown-menu-related components:

- `DropdownMenu`: The wrapper component provides context, state, and focus management.
- `DropdownMenuTrigger`: The trigger for the menu list. Must be a direct child of `DropdownMenu`.
- `DropdownMenuContent`: The wrapper for the menu items. Must be a direct child of `DropdownMenu`.
- `DropdownMenuArrow`: An optional arrow element to render alongside the menu content. Must be a direct child of a `DropdownMenuContent`.
- `DropdownMenuItem`: The trigger that handles menu selection. Must be a direct child of a `DropdownMenuContent`.
- `DropdownMenuGroup`: A wrapper to group related menu items.
- `DropdownMenuGroupLabel`: Used to render the label of a group. It won't be focusable using arrow keys.
- `DropdownMenuDivider`: A visual separator for menu items and groups.
- `DropdownMenuItemIndicator`: Renders when the parent `DropdownMenuCheckboxItem` or `DropdownMenuRadioItem` is checked. You can style this element directly, or you can use it as a wrapper to put an icon into, or both.
- `DropdownMenuCheckboxItem`: An item that can be controlled and rendered like a checkbox.
- `DropdownMenuRadioGroup`: Used to group multiple `DropdownMenuRadioItem`s.
- `DropdownMenuRadioItem`: An item that can be controlled and rendered like a radio.

```javascript
import {
  DropdownMenu,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuArrow,
  DropdownMenuGroup,
  DropdownMenuGroupLabel,
  DropdownMenuItem,
  DropdownMenuDivider,
} from '@resolid/nxt-ui';
```
