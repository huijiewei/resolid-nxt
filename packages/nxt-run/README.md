# @resolid/nxt-run

Get your app up and _running_ with [React](https://react.dev)!

## Install

```shell
## NPM
npm install @resolid/nxt-run
## PNPM
pnpm add @resolid/nxt-run
## Yarn
yarn add @resolid/nxtrun
```
